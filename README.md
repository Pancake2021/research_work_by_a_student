# research_work_by_a_student

Comparative study of PPO, GRPO, and modifications for LLM behavioral pattern analysis.

## Demo
![Demo](https://raw.githubusercontent.com/Pancake2021/Pancake2021/main/assets/cloud-drift.gif)

## How to run
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run_experiments.py
```

## Results
- Side-by-side comparison of policy optimization approaches.
- Training stability and sample-efficiency observations.
- Structured outputs ready for report and plot generation.
